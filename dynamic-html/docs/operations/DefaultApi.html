This is the API

recipesPost

Creates new recipe with related details, ingredients and procedures

URL

https://api.cipefy.com/v1/recipes

HTTP Method

POST

Response Type

Parameters

  • Form: title String

  • Form: description String

  • Form: servings Integer

  • Form: prep_time Integer

  • Form: prep_time_unit String

  • Form: cook_time Integer

  • Form: cook_time_unit String

  • Form: ingredientTitle String

  • Form: ingredientQuantity Integer

  • Form: ingredientUnit String

  • Form: procedures String

recipesRecipeDelete

Deletes recipe

URL

https://api.cipefy.com/v1/recipes/{recipe}

HTTP Method

DELETE

Response Type

Parameters

  • Path: recipe Integer

    ID of recipe to delete

recipesRecipePut

Updates recipe by ID

URL

https://api.cipefy.com/v1/recipes/{recipe}

HTTP Method

PUT

Response Type

Parameters

  • Path: recipe Integer

    ID of recipe to update

  • Form: title String

  • Form: description String

  • Form: servings Integer

  • Form: prep_time Integer

  • Form: prep_time_unit String

  • Form: cook_time Integer

  • Form: cook_time_unit String

  • Form: ingredientTitle List(Object)

  • Form: ingredientQuantity List(Object)

  • Form: ingredientUnit List(Object)

  • Form: procedures List(Object)
